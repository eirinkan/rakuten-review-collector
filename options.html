楽天レビュー収集

出力設定

空欄の場合はCSVモードで動作します

  1. Googleスプレッドシートを作成
  2. 拡張機能 → Apps Script を開く
  3. 以下のコードを貼り付け:
    gas/Code.gs の内容をコピー
  4. デプロイ → 新しいデプロイ
  5. 種類: ウェブアプリ
  6. アクセスできるユーザー: 全員
  7. デプロイ後のURLをコピーして上記に貼り付け

GASコードの更新方法

  1. スプレッドシート → 拡張機能 → Apps Script を開く
  2. Code.gs の内容を最新版に置き換え
  3. デプロイ → デプロイを管理 → 編集（鉛筆アイコン）
  4. バージョン: 「新バージョン」を選択
  5. 「デプロイ」をクリック

※URLは変わりません。コード更新後は必ず新バージョンをデプロイしてください。

ヘルプ

基本的な使い方

  1. 単一商品のレビュー収集:
    楽天市場の商品ページまたはレビューページを開き、拡張機能アイコンをクリック → 「収集開始」ボタンを押す
  2. 複数商品の一括収集:
    各商品ページで「キューに追加」→ この画面で「一括収集開始」
  3. ランキングから一括追加:
    楽天ランキングページのURLを入力 → 上位N商品を指定 → 「追加」
    または、ランキングページで拡張機能アイコンをクリック → 商品数を指定して追加

データの保存方法

  • CSVモード（デフォルト）: GAS URLが空の場合、CSVファイルとしてダウンロード
  • スプレッドシートモード: GAS Web App URLを設定すると、自動でGoogleスプレッドシートに保存

収集できるデータ

レビュー日、商品管理番号、商品名、商品URL、評価、タイトル、本文、投稿者、年代、性別、注文日、バリエーション、用途、贈り先、購入回数、購入情報、参考になった数、ショップ名、ページURL、収集日時

注意事項

  • 収集中はタブを閉じないでください
  • 楽天市場のサーバー負荷軽減のため、ページ間に3〜6秒の待機時間が入ります
  • 大量のレビューを収集する場合は時間がかかります
